# PancreaScan AI

**PancreaScan AI** is a full-stack healthcare application that leverages deep learning (**ResNet101**) to detect pancreatic cancer from CT scan images.

## Features

- **Early Cancer Detection**  
  Identifies potential pancreatic cancer at an early stage using AI-powered image analysis.

- **AI-Based Diagnostic Reports**  
  Generates detailed reports based on model predictions.

- **Confidence Scores & Risk Analysis**  
  Provides probability scores and risk insights to assist medical professionals.

- **Patient History Tracking**  
  Maintains patient records for continuous monitoring and comparison over time.

## Technology Stack

- **Frontend:** (React)
- **Backend:** (Flask)
- **Model:** ResNet101 (Deep Learning)
- **Database:** (MongoDB)

## Objective

To assist healthcare professionals in the early detection of pancreatic cancer and improve diagnostic accuracy using artificial intelligence.

## ⚠️ Disclaimer

This application is intended for **supportive diagnostic purposes only** and should not replace professional medical advice.
